## 📖 Abstract
This project presents a secure and scalable IoT data-sharing system using **Blockchain**, **Proxy Re-Encryption (PRE)**, and **Artificial Intelligence (AI)**.  

It ensures:
- Secure data encryption at the source  
- Decentralized access control using blockchain  
- Privacy-preserving data sharing  
- AI-based anomaly detection for enhanced security  

---

## 🚀 Features
- 🔒 End-to-End Data Encryption (ECC + AES)
- ⛓️ Blockchain-based Access Control (Immutable Ledger)
- 🔁 Proxy Re-Encryption for Secure Data Sharing
- 🤖 AI-Based Threat Detection (Isolation Forest)
- 📊 Real-time Dashboard using Streamlit
- 🌐 IoT Integration using ESP32

## 🏗️ System Architecture
The system consists of the following modules:

1. IoT Device Layer (ESP32 Sensors)
2. Gateway (Flask API)
3. AI Monitoring Module
4. Blockchain Manager
5. Proxy Re-Encryption Module
6. User Dashboard (Streamlit)

---

## 🔄 Workflow
1. IoT device collects and encrypts data  
2. Data sent to Flask gateway  
3. AI analyzes user behavior  
4. Blockchain verifies access  
5. PRE re-encrypts data for authorized user  
6. User views decrypted data on dashboard
